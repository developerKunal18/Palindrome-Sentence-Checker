# Palindrome Sentence Checker

A Python script that checks whether a sentence is a palindrome.

## Description
This project removes spaces, punctuation, and case differences before checking if the sentence reads the same forward and backward.

## Features
- Ignores spaces
- Ignores punctuation
- Case-insensitive comparison

## Technologies Used
- Python
- re module

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python palindrome_sentence.py

4. Enter a sentence when prompted

## Example

Input:
A man a plan a canal Panama

Output:
Palindrome
